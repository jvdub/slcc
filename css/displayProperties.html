Display Types

Using the display css attribute, you can instruct the browser how to show or hide the element.

These are some common options for the display attribute:

  • block
    • display: block; tells the browser to add new lines before and after the element.
  • inline
    • display: inline; tells the browser to show the element innnline with the other elements (no new lines).
  • inline-block
    • display: inline-block; tells the browser to display the element inline but to not ignore padding and margin.
  • none
    • display: none; tells the browser to hide the element.
    • Can also use visibility: hidden;
There are more options that are used for specific reasons that you can investigate if you'd like. I recommend that you note the grid option and look into that later.
