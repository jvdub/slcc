Bootstrap!

Over the last two weeks, we've learned a lot about CSS. In case you haven't noticed, there is a lot to learn about CSS and there are multiple ways to accomplish the same things. You might be wondering, "Is there an easier way? Arent' there people that have made this easier?"

I'm pleased to tell you, yes, there is an easier way.

That said, a basic understanding of CSS is critical. While tools others have written will get you up to speed quickly, it'll only get you so far. You'll almost always need to write your own CSS at one point.

So, without further ado, let's talk about that easier way.

Bootstrap

Boostrap is a CSS framework that takes care of a lot of the basic stuff for you.

Boostrap makes it easier to create responsive websites.

Bootstrap is a combination of CSS and JavaScript that makes your site responsive and interactive with little effort on your part.
