Positioning Elements

The css attribute position enables you to more specifically define the position of the element on the page.

These are the available options:

  • static
    • position: static; is the default. It's positioned in the normal flow.
  • relative
    • position: relative; causes the element to be in the normal flow and then offset relative to itself based on the values of top, right, bottom, and left.
  • fixed
    • position: fixed;
  • absolute
    • position: absolute; removes the element from the normal flow and is positioned relative to other positioned elements (if any).
  • sticky
    • position: sticky; causes the element to be positioned in the normal flow and then offset relative to the nearest scrolling ancestor.
