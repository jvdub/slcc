CSS Declarations

Let's discuss some of the most common style properties used.

Background

The background property is widely changed.

Let's face it: plain white backgrounds are pretty boring.

The most basic way to change the background is just change the color.

Example


                    <p style="background-color: cornflowerblue;">Example</p>

Oftentimes you'll also want to change the text color as well. Simply use the color property to do so.

Dark example.


                    <p style="background-color: midnightblue; color: white;">Dark example.</p>

The background-color style can be applied to any HTML element with a display area.

The color style can be applied to any HTML element with text in the body of the page.

Spacing (Box Model)

Don't stand so close to me

Spacing is used to put give HTML elements breathing room for a good user experience.

There are three main properties used for spacing:

      margin
      border
      padding

The first one to adjust is typically margin, though all play a part in good spacing.

Here's an example:

Gluten needs space!

Gluten needs space!


            <p style="background-color: wheat;">Gluten needs space!</p>
            <p style="background-color: wheat; margin: 30px;">Gluten needs space!</p>

You can use margin-<SIDE> to apply margin to a specific side of the element.

Or you can use margin: top right bottom left; to apply different values to different sides all at once.

            <p style="background-color: wheat; margin: 10px 50px 25px 100px;">All around</p>
            <p style="background-color: wheat; margin: 10px 50px 25px 100px; padding: 10px; border: solid 2px black;">All around</p>

Units of Measure

There are various ways to tell the browser how far to move things or how much room it should occupy.

Pixels, or px, is a very common way but is not best practice.

Pixels are too rigid and don't translate well to devices with different screen sizes and pixel desities.

This guide does a great job explaining the current best practices.

Height and Width

Height and width are critical properties to use.

The browser will default to the minimum height and minimum width needed for an element to be displayed.

At times, that's just not enough.

If you want to center text on the page or on a section of the page, you'll need to element (and possibly its parents) to take up the entire width and height. You do this with the width and height properties.

Example:


            <div style="background-color: tomato; height: 100px; width: 100px;"></div>

Any valid CSS unit is acceptable here (%, em, rem, etc.)

Can also specify minimum and maximum height and width with their accompanying attributes:

  • min-height
  • max-height
  • min-width
  • max-width

Font and Text

Font is a very common thing to change.

From colors to font face to size and more, this is one of the key parts of any website.

Here are the basic font properties:

  • font-family
    • This changes the font of the text (eg. Times New Roman, Arial, Comic Sans, etc.)
  • font-size
    • This changes the size of the font (usually denoted with em)
  • font-style
    • Can denote normal, italic, or oblique.
  • font-weight
    • Can be normal, bold, or a number denoting how bold to make the text.
  • color
    • A simple color property of the text (any valid CSS color is accepted)
  • text-align
    • Used to align the text left, right, center, or justified.

All the things!


                    .all-the-things {
                        font-family: Cambria, Cochin, Georgia, Times, 'Times New Roman', serif;
                        font-size: 1em;
                        font-style: italic;
                        font-weight: bold;
                        color: #0045CC;
                        text-align: center;
                    }

                    <p class="all-the-things">All the things!</p>
