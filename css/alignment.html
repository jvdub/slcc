Aligning Elements & Content within Elements

Aligning elements can be accomplished in a number of ways.

To align elements and other content horizontally, use margin: auto;

To align images, you'll need to set the display property as well: 


                img {
                    display: block;
                    margin-left: auto;
                    margin-right: auto;
                }

To center text, use text-align: center;.

To centering vertically is most commonly accomplished using padding.
