Float Property

The float property places the element on the left or right side of its container allowing other elements to wrap around it.

These are the options:

  • left
    • float: left; puts the element on the left within its parent
    • float: right; puts the element on the right within its parent

You may occasionally run into issues where a floated element exceeds the length of the parent element and disrupts the flow of later elements.

To fix this, us the clearfix class:


            .clearfix:after {
                content: "";
                clear: both;
                display: table;
            }

The clear property is the main one here. It clears the left and right floats of preceeding elements.
