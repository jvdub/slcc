Pseudo-Classes

Pseudo-classes in CSS are keywords that specifies a special state of the selected element(s).

The most common might be :hover.

You can find a list of the most common Pseudo-classes here.

Here are some other common ones:

  • :hover
    • :hover defines the styles for when the cursor is over the selected element(s)
  • :focus
    • :focus defines the styles for when the user has selected the specified element(s)
  • :active
    • :active defines the styles for when the user is clicking on the selected element(s)
  • :visited
    • :visited defines the styles for when the user has visited the selected link(s)
