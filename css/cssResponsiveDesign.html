CSS/Website Responsive Design

Activity #1: Working with CSS file and HTML Document

In this activity you will create your own .css file and link it to your HTML document. You will also set specific styles to elements throughout your document.

Step 1
Create the .css file and link it to your HTML document.
Step 2
Create a style map on a piece of paper. Decide placement for each bit of content, and stylistic elements such as color, font, backgrounds, etc.
Step 3
Create element style statements for all major elements.

Set styles for the following:

  • Headings
  • Paragraphs
  • Body
  • Header
  • Footer
  • Nav
  • a tags
  • img tags

Step 4
Create classes for any specialty design elements. Classes are defined with a name following a '.'
Step 5
Save and upload both your CSS file and your HTML document.

  • Create new version with changes using GIT commit command
  • Upload content to the webserver using FTP
  • View your web page in a browser to ensure it looks the way you want
  • Go back and tweak CSS as needed to fine tune and republish
