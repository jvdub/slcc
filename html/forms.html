HTML Forms

HTML form are a way to send data to the server.

HTML forms are used all over the internet. Many times, if you're entering information in text field or making selections with radio or checkboxes, you're filling out a form.

HTML forms are created usind the <form> tag.

HTML forms have three key parts:

  1. The input fields (e.g. text boxes, checkboxes, etc.)
  2. A submit button
  3. An action (where to send the data)

Here's a quick example of a form: 


            <form action=""> <!-- You'd normally have an action defined here. For this small example, I left it blank. -->
                <label>
                    Please enter your name:
                    <input type="text" name="name">
                </label>
                <button type="submit">Submit</button>
            </form>
