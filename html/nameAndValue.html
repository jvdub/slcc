Names and Values

Names and Values help define the data to be passed to the server.

Each input type can have a name and a value attribute.

If you don't definet he value attribute, one will be added behind-the-scenes in HTML.

The name attribute should be included on all input fields that are in a form

Here's an example:


            <form>
                <label>
                    Username:
                    <input type="text" name="username" value="">
                </label>
                <label>
                    Password:
                    <input type="password" name="password" value="">
                </label>
                <input type="submit" name="login" value="Log In">
            </form>
