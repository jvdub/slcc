Tables

Let's learn about HTML tables!

Tables are denoted with the <table> tag.

There are three parts to a table: rows, headers, and data

Rows are denoted by the <tr> tag.

Header cells are denoted by the <th> tag.

Data cells are denoted by the <td> tag.

Let's put this all together. First the code:


            <table>
                <tr>
                    <th>Name</th>
                    <th>Color</th>
                </tr>
                <tr>
                    <td>John</td>
                    <td>Blue</td>
                </tr>
                <tr>
                    <td>Jane</td>
                    <td>Red</td>
                </tr>
                <tr>
                    <td>Alice</td>
                    <td>Green</td>
                </tr>
                <tr>
                    <td>Bob</td>
                    <td>Black</td>
                </tr>
            </table>

And here's what it looks like:

Name Color
John Blue
Jane Red
Alice Green
Bob Black

You can add a title to a table using the <caption> tag.

Sometimes you need cells to take up multiple columns or multiple rows. You can accomplish this with the colspan and rowspan attributes

This is a caption
Header
Side Bar Name Color
John Blue
Jane Red 

            <table>
                <caption>This is a caption</caption>
                <tr>
                    <th colspan="3" style="background-color: lightblue">
                        Header
                    </th>
                </tr>
                <tr>
                    <th rowspan="3" style="background-color: bisque;">
                        Side Bar
                    </th>
                    <th>Name</th>
                    <th>Color</th>
                </tr>
                <tr>
                    <td>John</td>
                    <td>Blue</td>
                </tr>
                <tr>
                    <td>Jane</td>
                    <td>Red</td>
                </tr>
            </table>
