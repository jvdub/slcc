Creating an HTML Table

In this activity, you will organize your references and/or testimonials into a table to keep them organized and displayed in a professional way.

Step 1
Map out on paper the table you want to create so you know how many rows and columns you need
Step 2
Create the table with appropriate tags to layout as you have mapped

Hint: Use all four main tags <table> <th> <tr> <td>

Step 3
Stylize the table to look good with the rest of the site

Hint: Use styles learned from last week within specific table tags

Step 4
Save and upload your file
