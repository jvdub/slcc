Images

Let's learn about images!

Images are a very common element of a webpage. You can add images to your web page using the <img> tag.

The src attribute tells the browser where to look for the image.

Another useful attribute is the title attribute. It is what is displayed if the image cannot be found or for the visually impaired.

Let's add an image to this page!


            <img src="../images/coronavirus.jpg" title="Coronavirus">

Other useful attributes are the height and width attributes. These attributes help you size the iamge so that it appears like you want it to.


            <img src="../images/coronavirus.jpg" title="Coronavirus" height="155" width="225">
