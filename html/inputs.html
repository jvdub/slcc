HTML Input Types

HTML inputs are how users enter data.

There are many types of inputs. We'll cover many of them here.

All the inputs discussed in this section use the <input> tag.

Most inputs are more helpful when they have a label. You can add a label to an input in one of two ways.

Surrounding the input:


            <label>Please enter your name:
                <input type="text">
            </label>

Or by using ID's:


            <label for="name">Please enter your name:</label>
            <input type="text" id="name">

Text

Text inputs are among the most common iputs used.

They're used to gather basic text data from the user.

You create this type of input with the following tag: <input type="text">

Here's an example:


            <input type="text"> Test
        

            <label for="name">Please enter your name:</label>
            <input type="text" id="name">

Radio

Radio inputs are used when you want users to select one and only one of the available options.

You create this type of input with the following tag: <input type="radio">

Here's an example:

Test 

            <input type="radio">

Radio buttons aren't really useful without more than one button and labels to distinguish them.


            <label>
                <input type="radio" name="radioOption" value="1"> Option 1
            </label>
            <label>
                <input type="radio" name="radioOption" value="2"> Option 2
            </label>

Checkbox

Checkboxes are very similar to radios but they allow multiple options to be selected.

You create this type of input with the following tag: <input type="checkbox">

Here's an example:

Test 

            <input type="checkbox"> Test

While checkboxes can have a single option, they're often found in groups with labels.


            <label>
                <input type="checkbox" name="checkboxOption" value="1"> Option 1
            </label>
            <label>
                <input type="checkbox" name="checkboxOption" value="2"> Option 2
            </label>

Password

Password input types are usually used on login or account creation pages.

They're very similar to text inputs but mask the data as it's entered.

You create this type of input with the following tag: <input type="password">

Here's an example:


            <label>
                Password:
                <input type="password">
            </label>

Submit

The submit input type creates a button that submits the form that contains it.

You create this type of input with the following tag: <input type="submit">

Here's an example:


            <input type="submit">

You can customize the text of the button with the value attribute as follows:


            <input type="submit" value="Send it!">

Date

The date input type restricts the user to selecting a valid date.

You create this type of input with the following tag: <input type="date">

Here's an example:


            <input type="date">

Using the min and max attributes, you can restrict the dates the user can select from.


            <label for="start">Start date:</label>
            <input type="date" id="start" name="trip-start" value="2020-07-22" min="2020-01-01" max="2020-12-31">

Email

The email input restricts the user to entering a valid email address.

You create this type of input with the following tag: <input type="email">

Here's an example:


            <input type="email">

Number

The number input restricts the user to enter a valid number.

You create this type of input with the following tag: <input type="number">

Here's an example:


            <input type="number">

Using the min and max attributes, you can restrict the dates the user can select from.


            <label>
                Favorite Number:
                <input type="number" min="0" max="100">
            </label>

URL

The URL input type restricts the user to a valid URL.

You create this type of input with the following tag: <input type="url">

Here's an example:


            <input type="url">

Color

The color input allows the user to select a color from a color pallette.

You create this type of input with the following tag: <input type="color">

Here's an example:


            <input type="color">
