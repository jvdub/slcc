Links

Let's learn about Links!

Links are what gave HTML part of its name. They called it "Hyper-text" because it could link to other text files thus making it much more capable than your typical text document.

Links are created using the anchor tag: <a>

The links most important attribute is the href attribute. You use this attribute to tell the link where to navigate to when the user interacts with it.

Here's an example:

Google


            <a href="https://www.google.com">Google</a>

You can use the target attribute to tell the browser how to open the link. Here are the possible options:

Value Description
_blank Opens the linked document in a new window or tab
_self Opens the linked document in the same frame as it was clicked (this is default)
_parent Opens the linked document in the parent frame
_top Opens the linked document in the full body of the window
framename Opens the linked document in a named frame
