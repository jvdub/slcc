Special Inputs

Sometimes you need a little something more.

There are three types of special inputs in HTML. Let's talk about them!

Select

Select inputs create a dropdown menu or a multi-select box.

Each item in the list is an "option" created with the <option> tag.

Here's an example:


            <select>
                <option>Option 1</option>
                <option>Option 2</option>
            </select>

And here's a multi-select:


            <select multiple>
                <option>Option 1</option>
                <option>Option 2</option>
            </select>

Textarea

The <textarea> tag creates a multi-line text input field.

This is used when the user needs to enter a paragraph (or more).

Here's an example:


            <textarea></textarea>

You can define the initial height and width with the rows and cols attributes


            <textarea rows="5" cols="50"></textarea>

Button

The <button> tag is used to create a button.

Buttons created with this can either be submit buttons on forms or other user interactions.

Here's an example:


            <button type="button" onclick="alert('You clicked me!');">Don't click me</button>
