Working With Images and Links

In this activity, you will

  • Add a profile image to your HTML file and format it to fit the file
  • Create a menu that goes to bookmarked location on your resume when clicked
  • Add links to a downloadable version of your resume and to past employers’ websites and stylize them
Step 1
Add your profile image to your resume site

Hint: Use img tags and define the widthor height to fit the image nicely into the appropriate element

Step 2
Create a menu within the nav tag with section names
Step 3
Add ids to the sections you would like the viewer to be taken to when clicking on the bookmark links

Hint: Make the ids match the menu titles

Step 4
Create appropriate links to direct viewers to each bookmark when they click on a menu item

Hint: Use # followed by the section id to refer to the location on the site

Step 5
Create a link to a downloadable pdf ofyourresume that opens in a new window

Hint: Upload the pdf file to your domain before it is a functioning link

Hint: Define the target to make sure the link opens in a new window

Step 6
Create links to your previous/current employers’ websiteslisted inyour resume

Hint: Usethe entire URL

Step 7
Save and upload your web site to your domain via Filezilla
