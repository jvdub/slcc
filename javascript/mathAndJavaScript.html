Math Methods

JavaScript comes with a number of mathematical functions built-in using its Math object.

NOTE: It is case-sensitive meaning the "M" must be upper-case.

Here are the methods on the Math object:

  • .round
    • This method rounds the number to the nearest whole number. (eg. 1.5 goes to 2, 1.4 goes to 1)
  • .ceil
    • This method increases the number to the nearest whole number. (eg. 1.5 goes to 2, 1.4 goes to 2, 1.1 goes to 2, 1 stays at 1)
  • .floor
    • This method decreases the number to the nearest whole number. (eg. 1.5 goes to 1, 1.4 goes to 1, 1.9 goes to 1, 2 stays at 2)
  • .pow
    • This method raises the number to the given power (exponent) (eg. Math.pow(2, 2) is 2 to the power of 2, Math.pow(3, 9) is 3 to the power of 9)
  • .sqrt
    • This method takes the square root of the given number (eg. Math.sqrt(2) calculates the square root of 2)
  • .min
    • This method returns the lowest number in the given list or array. (eg. Math.min([1, 2, 3, 4, 5]) returns 1)
  • .max
    • This method returns the highest number in the given list or array. (eg. Math.min([1, 2, 3, 4, 5]) returns 5)
  • .random
    • This method returns a random number from 0 to 1. Often multiplied by another number to get a random number in the range you want. (eg. Math.random() * 10 generates a random number between 0 and 10)
