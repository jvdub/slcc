JavaScript Output

So... How does JS interact with the web page?

There are multiple ways to interact with the web page! Let's talk the most common.

innerHTML

innerHTML is used to set the HTML within the given element.

This is possibly the most common way to output to the page in JavaScript.


                <p id="my-paragraph"></p>
                <script>
                    document.getElementById('my-paragraph').innerHTML = 'The text for my paragraph';
                </script>

document.write

This method is less common. This method is used to write an entire page.

Generally this is only used when creating entire web pages in JavaScript.

Here is a good eample of this.

window.alert()

This is a once common method to create pop-up windows.

With the advent of more powerful CSS and HTML, window.alert() is not used as much anymore.

Bootstrap offers another way to create pop-up windows with its .modal() functions and classes.

console.log()

console.log() is a very common method to output to the developer tools.

console.log() often used when testing and debugging to see values at different points of execution.

Keep console.log() in mind and use it as you develope site. Just remember to remove unnecessary or sensitive logs before going live.
