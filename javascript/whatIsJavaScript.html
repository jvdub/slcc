Coding

Coding is creating instructions, or code, that the machine understands.

JavaScript is a programming language that enables you to create instructions for the computer to execute.

JavaScript is unique in that it's one of the only languages that web browsers can run.

JavaScript is a "Client-Side" scripting language. This means that the JavaScript you write will eventually run in a web browser on someone else's computer.

Because JavaScript is designed to run in the web browser, it offers a number of built-in functions that help interact with the web page. JavaScript can even create HTML elements and write to the web page.

To get started using JavaScript, all you need to do is insert a <script> tag.


                <script>
                    document.write('Hello World!');
                </script>

While you can write JavaScript within the script tag as shown above, it's usually best practice to put your JavaScript in separate files.


                <script src="myJavaScript.js"></script>
