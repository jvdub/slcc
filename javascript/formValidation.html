Form Validation

Improving UX and security

Form validation has two main purposes:

  • Make sure the data is what you think it is
  • Help the user enter the data they need to enter

Form validation can happen in two places: on the user's machine in the browser and on the server after they submit the data.

Both client-side and server-side validation are essential!

Today we'll be talking about client-side validation.

Client-Side Validation

Client-side validation uses a combination of HTML, CSS, and JavaScript to validate the data and indicated to the user the results.
