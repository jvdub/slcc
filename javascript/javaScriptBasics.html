JavaScript Basics

JavaScript Code

You've seen some examples of code. Let's go over some syntax.

Here is a basic statement:


                let myVariable = 123;

Here we declare a variable using the let keyword. We name the variable myVariable and assign the value 123 to it. Finally, we terminate the statement with the semi-colon (;).

Variables are used to store information that the program can use. Variables are so named because their contents can vary over the course of the program.

JavaScript also has constants. These are similar to variables in that they store data for the program, but their contents don't change.

Variable Types

There are five types of variables in JavaScript:

  • Number (eg. 1, 2, 100, 200, etc.)
  • String (eg. 'Hello World!')
  • Object (eg. { make: 'Nissan', model: 'GT-R', year: '2020', drive: 'AWD', hp: 600 })
  • Boolean (eg. true, false)
  • Null (eg. Null, basically means the variable is defined but holds nothing)

Types of Operators

Operators allow us to perform specific actions with our variables.

Here are the four types of operators:

  • Arithmetic (add, subtract, multiply, etc.)
  • Comparison (equal, not equal, greater than, etc.)
  • Assignment (Store this value in this variable)
  • Logical (And, or, etc.)

Functions

Functions are a way to bundle up a set of instructions to use over and over or to organize specific actions.

Here is the basic syntax of a function:


                function myFunction() {
                    document.write('Test this out!');
                }

                myFunction(); // This is how to call the function

Objects

Objects are heavily used in JavaScript and are highly useful.

Objects are a collection of values that describe an item meaningful to the program.


                let person = {
                    firstName: 'John',
                    lastName: 'Van Wagenen',
                    phone: '801-555-5555',
                    email: 'test@email.com',
                    address: {
                        street: '1234 test parkway',
                        city: 'Anytown',
                        state: 'UT',
                        country: 'USA'
                    }
                };

                console.log(person.firstName); // prints 'John'
                console.log(person.lastName); // prints 'Van Wagenen'
                console.log(person.address.city); // prints 'Anytown'

Objects can hold many things and many types of values. They are very flexible and can represent just about any data you'll need.

Arrays

While objects allow you to represent complex data objects, Arrays allow you to store multiple similar things.


                let breakfastFoods = [
                    'yogurt',
                    'granola',
                    'pancakes',
                    'eggs',
                    'bacon',
                    'quiche'
                ];

                console.log(breakfastFoods[0]); // prints 'yogurt'
                console.log(breakfastFoods[1]); // prints 'granola'
