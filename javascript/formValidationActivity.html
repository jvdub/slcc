Form Validation

Use HTML, CSS, and JavaScript to validate the contact form.

Step 1

Decide which fields to make required, which ones to restrict to a pattern, and so forth.

Step 2

Add the necessary attributes to each field.

  • required to make the field required
  • pattern="..." to restrict it to a pattern
  • min and max and minlength and maxlength as needed

Step 3

Add the necessary CSS to indicate which fields are invalid.

Step 4

Decide if any fields cannot be validated with the built-in HTML attributes. Design JavaScript functions to validate these fields or scenarios.

Step 5

Implement and test these JavaScript functions.

Step 6

Test that the form validates as you planned.

Step 7

Save and upload your site via filezilla (or desired web browser)
