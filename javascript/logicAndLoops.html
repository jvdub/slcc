Conditional Statements

Using comparison and logical operators, you can now formulate your own conditional statments.

Conditional Statements are generally created with the if keyword. The if keyword can be used in conjunction with the else keyword. You can even chain them together and have else if statments as well.


            if (doIHaveAThing() === true) {
                // If doIHaveAThing returns true, the code in here will run
                console.log('I have a thing!');
            } else if (doIHaveAThing() === true && doesMegamindWantsMeToLeave() === true) {
                // If I have a thing and Megamind wants me to leave, this code block will run
                console.log("Fine! I'll pack my thing and leave!");
            }
            else {
                // If I don't have a thing, this code block will execute
                console.log('I wish I had a thing...');
            }

Loops

Loops make doing repetative things tolerable.

With a loop, you can perform a certain set of instructions multiple times while only writing those instructions once.

Let's consider the case where you need to display a list of items on the screen. Here's our list:


                    let myList = [ "HTML", "CSS", "JavaScript", "PHP", "MySQL" ];

while

The while loop is the most basic loop. To display the items in the list above, we'd do something like the following:


                    let i = 0;
                    while (i < myList.length) {
                        console.log(myList[i]);
                        ++i;
                    }

This will log every item in the array to the console and then stop when the index is bigger than the length of the array.

What happens if we increment i before logging myList[i]?

NOTE: While loops are basic and they're very easy to create infinite loops with. Make sure your condition can and will be false at some point.

NOTE: The condition in the parens is treated as any if statement. It can be as complex or simple as you'd like (please keep it simple).

do/while

The do/while loop is similar to the while loop except it will always execute at least once.


                    let i = 0;
                    do {
                        console.log(myList[i]);
                        ++i;
                    } while (i < myList.length);

This will log every item in the array to the console and then stop when the index is bigger than the length of the array.

What happens if we increment i before logging myList[i]?

for

The for loop is a shorthand way of writing a while loop.


                    for (let i = 0; i < myList.length; ++i) {
                        console.log(myList[i]);
                    }

This will log every item in the array to the console and then stop when the index is bigger than the length of the array.

for/in

The for/in loop is similar to the for> loop except it takes care of the counting for you.


                    for (let item in myList) {
                        console.log(item);
                    }

This will log every item in myList to the console.

for/of

The for/of loop is very similar to the for/in loop except it only iterates over iterable items and not all string-typed keys (like for/in).


                    for (let item of myList) {
                        console.log(item);
                    }

This will log every item in myList to the console.

Whenever possible, use for/of. As you get to know JS more, learn the why behind this. For now, just use it by default if at all possible.

Special Keywords

Sometimes when looping, you want to either stop looping altogether once something happens or you want to skip certain items in the list.

The break and continue keywords make this possible.

  • break
    • Stops the loop altogether and begins executing the code immediately following the loops closing brace.
  • continue
    • Skips the current iteration of the loop and continues to the next iteration of the loop.
